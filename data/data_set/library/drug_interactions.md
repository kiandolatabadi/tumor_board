# Drug Interaction Reference (Internal, Fictional canned lookup)
*Illustrative entries for demo purposes, not clinically validated. Read by `check_drug_interactions`.*

## INT-001 — Tamoxifen + strong CYP2D6 inhibitors
Co-administration with strong CYP2D6 inhibitors (e.g., paroxetine, fluoxetine, bupropion) may reduce conversion to active endoxifen, blunting therapeutic effect. **Moderate severity.**

## INT-002 — Capecitabine + Warfarin
Capecitabine potentiates warfarin's anticoagulant effect via CYP2C9 inhibition, risking significant INR elevation and bleeding. Capecitabine dosing itself is renally adjusted, so **a current (within 30 days) creatinine/eGFR is required before this combination can be cleared** — an outdated renal panel means the interaction cannot be safely assessed, not that it's absent. **Severity: high; renal-function-dependent.**

## INT-003 — Ribociclib + QT-prolonging agents
Ribociclib prolongs QTc; co-administration with other QT-prolonging agents (e.g., ondansetron, certain antibiotics) requires baseline and on-treatment ECG monitoring. **Moderate severity.**

## INT-004 — NSAIDs (e.g., ibuprofen) + antiplatelet/anticoagulant therapy
Additive bleeding risk; generally low-acuity in the absence of therapeutic anticoagulation. **Low severity — informational only.**
